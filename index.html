How to use

  1. Open the document in Word and open the Copilot pane.
  2. Click Copy prompt above and paste it into Copilot.
  3. Check the output against the guardrails: if it flags gaps, that is working as intended.
  4. If anything looks off, reply: "Re-check point 2 against the source and quote the exact text it comes from."
Why Copilot summaries go wrong (and how this fixes it)
  • Vague ask: "Summarize this" lets Copilot pick its own scope and depth. Fix: explicit audience, length and sections.
  • Hallucinated detail: Copilot fills gaps with plausible-sounding text. Fix: source-only rule + "Not stated in the source".
  • Mangled numbers: Figures get rounded or paraphrased. Fix: verbatim rule.
  • Long docs truncated: Copilot may only deeply read part of a very long file. Fix: the long-document mode asks for a section-by-section pass first.
  • Format drift: Without a schema it outputs walls of text. Fix: a fixed heading/table structure.
Pro tips for better Copilot results
  • In Copilot Chat, attach the file directly (paperclip or type / and pick the file) instead of describing it.
  • One task per prompt. Summary first, then follow-up email as a second prompt.
  • If Copilot answers from general knowledge, reply: "Only use the attached file. Discard everything else."
  • For scanned PDFs with poor text, Copilot may see garbage. Check the PDF has selectable text.
